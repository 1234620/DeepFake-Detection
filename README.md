# 🔍 DeepFake Detection - AI-Powered Media Authentication

A modern, production-ready web application for detecting deepfakes and manipulated media. Built with TypeScript React frontend and Python ML backend, ready to deploy on Render.

## ✨ Features

- **🎨 Modern UI Design** - Beautiful blue gradient interface (replacing purple)
- **⚡ Real-time Detection** - Fast AI model inference on GPU
- **📁 Easy Upload** - Drag & drop or click to upload images/videos
- **📊 Confidence Scores** - Detailed detection results with metrics
- **📱 Responsive Design** - Works perfectly on desktop, tablet, mobile
- **🔒 Secure** - Files processed in-memory, never permanently stored
- **🌐 Web-based** - No installation needed, works in any browser
- **☁️ Cloud Ready** - One-click deployment to Render

## 🚀 Quick Start

### Local Development (2 minutes)

```bash
# 1. Install dependencies
npm install
pip install -r requirements.txt

# 2. Start frontend (Terminal 1)
npm run dev
# → http://localhost:5173

# 3. Start backend (Terminal 2)  
python app.py
# → http://localhost:5000
```

Visit http://localhost:5173 and start detecting deepfakes! 🎉

## 📊 API Endpoints

### `/` (GET)
Serves the main frontend application.

### `/api/detect` (POST)
Main detection endpoint.

**Request:**
```bash
curl -X POST http://localhost:5000/api/detect \
  -F "file=@image.jpg"
```

**Response:**
```json
{
  "success": true,
  "is_deepfake": false,
  "confidence": 0.95,
  "processing_time": 2.34
}
```

## 🚀 Deployment

### Render


## 🐛 Troubleshooting

### Frontend Not Loading
```bash
npm run build
git push  # If deployed on Render
```

### API Connection Error
- Check that backend is running: `python app.py`
- Verify API URL in `src/config.ts`
- Check browser console for CORS errors

### Model File Not Found
- Ensure `deepfake_detector_best.pth` exists in project root
- For large files (>100MB), consider using Git LFS:
  ```bash
  git lfs install
  git lfs track "*.pth"
  git add .gitattributes
  git commit -m "Add LFS tracking"
  ```

### Render Build Timeout
- PyTorch installation takes time (~5 minutes)
- Consider upgrading to paid Render instance
- Use GitHub Actions to pre-build and cache

### Out of Memory Error
- Render free tier has 512MB
- Upgrade to Standard instance for better performance
- Optimize model size if needed
